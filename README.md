# Contract Analysis & Risk Assessment Bot

A GenAI-powered legal assistant for Indian SMEs that analyzes contracts,
identifies risky clauses, and explains them in simple business language.

## Features
- PDF / DOCX / TXT upload
- Clause-level risk scoring (Low / Medium / High)
- Plain-language explanations
- SME-friendly renegotiation suggestions
- Clean animated UI using Streamlit

## Tech Stack
- Python
- Streamlit
- Rule-based NLP + Generative AI reasoning

## How to Run
pip install -r requirements.txt
streamlit run app.py
